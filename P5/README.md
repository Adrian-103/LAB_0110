# Práctica 5

## Introducción
Esta práctica tuvo como objetivo el uso de modos de bajo consumo en **STM32**. En esta práctica la tarjeta entrará en un modo de ahorro de energía después de haber hecho una tarea específica.

## ¿Qué deberías de ver?
La tarjeta parecerá que se apaga una vez el microcontrolador llegó al estado correspondiente después de haber presionado el botón las veces necesarias. Despertará después de 10 segundos.
## Observaciones:
- Se usaró una LDR pero cualquier sensor o componente que pueda proporcionar un valor analógico funciona
- La terminal se configura desde antes en STM32CubeIDE, de otra forma no se podrán ver los datos sin un programa externo.
- Se usó la NUCLEO-H533RE
