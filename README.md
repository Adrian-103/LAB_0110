# LAB_0110
En este repositorio están los archivos de las prácticas de 5° semestre de sistemas embebidos, adaptadas para la tarjeta de desarrollo **NUCLEO-H755ZI-Q** y en algunos casos para la **NUCLEO-H503RB**. Ambas tarjetas son **STM32** y pertenecen a la **familia H**, es decir *High performance*.
*Todas las prácticas en este repositorio están basadas en STM32, que cuentan con una arquitectura ARM.*

En su mayoría los archivos funcionan para el core *Cortex-M7* del **H755ZI-Q**. Hay casos aislados en el que el M4 llega a ser utilizado. Intentaré poner la carpeta entera de las prácticas para poder tener una mayor comprensión de las prácticas, pero si algo sale mal estarán los archivos ```main.c``` de cada práctica.
